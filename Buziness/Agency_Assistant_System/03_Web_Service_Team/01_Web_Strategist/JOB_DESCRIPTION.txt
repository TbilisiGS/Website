Web Strategist Job Description

Main responsibilities:
- Create the sitemap and page hierarchy.
- Shape positioning, offers, packages, and calls to action.
- Define user journeys from first visit to inquiry.
- Recommend trust signals, proof sections, and conversion elements.
- Align messaging with the agency's premium positioning.

Success looks like:
- The website is clear, persuasive, and easy to navigate.
- Visitors understand the services quickly.
- The site converts strong-fit leads into inquiries.
