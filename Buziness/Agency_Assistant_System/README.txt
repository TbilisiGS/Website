Agency Assistant System

This folder gives you a small agency org chart made of AI assistants.
Each worker has a dedicated folder with:
- PURPOSE.txt
- JOB_DESCRIPTION.txt
- PROMPT.txt

How to use:
1. Open the worker folder.
2. Copy the text from PROMPT.txt into a new chat.
3. Add your specific task under the prompt.
4. Let that assistant focus only on its role.

Recommended order:
1. Start with the CEO Assistant for priorities.
2. Use the Operations Manager to turn ideas into plans.
3. Use the Web Strategist and Marketing Strategist to shape the offer.
4. Use the creative and web workers to produce assets and pages.
5. Use the Sales Client Manager to convert leads into paying clients.

Business focus:
We are building a premium services website for graphic design, video editing, and website creation services. The brand should feel high-value, modern, reliable, and conversion-focused.
